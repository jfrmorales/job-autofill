🔑 La API key se guarda solo en este navegador (chrome.storage.local) y se usa desde el background para llamar al proveedor que elijas. No la compartas. Si la expusiste en algún sitio, regénerala en el panel del proveedor.
Cada proveedor trae modelos sugeridos; «Personalizado…» acepta cualquier id válido de ese proveedor. Si agotas la cuota de un modelo, cambia a otro o de proveedor.
Opciones avanzadas Más baja = respuestas más deterministas. Por defecto 0.4. Súbelo si las respuestas se cortan. Por defecto 8192. Mejora la fiabilidad del JSON, pero algunos servidores compatibles lo rechazan: desactívalo si da error 400.
Nombre, email, LinkedIn, salario, etc. Lo usa la IA para rellenar los campos. El navegador no deja adjuntarlo por script, pero el popup te lo baja a Descargas con un clic para que solo tengas que seleccionarlo.
Al adjuntar un PDF arriba se extrae aquí automáticamente. La IA redacta cover letters y respuestas a partir de este texto, y rellena con él los campos de «Resume». Puedes editarlo.
